Demo URL : 
https://restaurant-yummy-yum.vercel.app/

Description:

Online restaurant website offering a smooth user experience with a modern, responsive design and Dark Mode support.
with the following features:

User:
1.	Fully Responsive Design (Mobile, Tablet, Desktop) via Tailwind CSS and flowbite.
2.	Dark & Light Mode.
3.	State Management Using Context and React-Query.
4.	Users Authentication and Validation By JWT and Google OAuth.
5.	Online food ordering system.
6.	Online Table Reservation.
7.	User review system. 
8.	Cart system with Stripe-based online payments.
9.	Restaurant Branches browsing.
    
Admin Dashboard:
1.	Admin can add, delete, and update categories, foods, branches, tables in every branch.
2.	Admin can Track and follow up on guest orders in real-time.
3.	Super Admin can add and delete admins at the system.
